# 🧠 17. AI Agent Instructions

# Rules for Coding Agent

## PRIORITY RULES

### Rule 1

Always build backend-first for critical systems.

---

### Rule 2

SOS system must be stable before AI integration.

---

### Rule 3

Realtime architecture must use WebSockets.

---

### Rule 4

Never expose location APIs without authentication.

---

### Rule 5

Use modular scalable architecture only.

---

### Rule 6

Use environment variables for all secrets.

---

### Rule 7

Every API must include:

* Validation
* Error handling
* Logging
* Authentication middleware
